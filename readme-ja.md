# 流体解析ダッシュボード

数値流体力学（CFD）解析用の対話型可視化ダッシュボードです。

## 機能

- 計算格子の可視化
- 時間発展データのプロット
- 速度場解析
- 圧力分布表示
- 発散（ダイバージェンス）の可視化

## 使用方法

1. ブラウザでダッシュボードを開く
2. 以下の解析ファイルをアップロード：
   - 時間発展データ（.dat）
   - 解析結果UVPデータ（.dat）
   - 発散データ（.dat）
   - 格子情報（.dat）
3. タブを切り替えて各種可視化結果を確認

## 依存ライブラリ

- Plotly.js（CDN経由でロード）
